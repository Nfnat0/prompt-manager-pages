# Prompt Manager Support

Prompt Manager helps you save, organize, search, and copy reusable AI prompts.

## Frequently Asked Questions

## Do I need an account?

No. Prompt Manager does not require an account.

## Where is my prompt data stored?

Your prompt data is stored locally on your device.

## How do variables work?

Add placeholders such as {% raw %}`{{topic}}`{% endraw %} to a prompt. When you copy that prompt, the app asks for values and copies the resolved text without changing the saved template.

## How do I contact support?

Contact: nfnat0.support@gmail.com.

Please include your iOS version, app version, device model, and a short description of the issue.

# Prompt Manager サポート

Prompt Manager は、再利用するAIプロンプトを保存、整理、検索し、すばやくコピーするためのアプリです。

## よくある質問

## アカウントは必要ですか？

いいえ。Prompt Manager はアカウント登録なしで利用できます。

## プロンプトデータはどこに保存されますか？

プロンプトデータはユーザーの端末内に保存されます。

## 変数はどのように使いますか？

プロンプト内に {% raw %}`{{topic}}`{% endraw %} のようなプレースホルダーを追加します。そのプロンプトをコピーするときに値を入力すると、保存済みテンプレートを変更せずに置換済みのテキストをコピーできます。

## サポートへの連絡方法

連絡先: nfnat0.support@gmail.com

問い合わせ時は、iOS バージョン、アプリバージョン、端末モデル、問題の概要を含めてください。
